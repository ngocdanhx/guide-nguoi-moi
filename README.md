# guide-nguoi-moi

## Đặc điểm

- Viết cho người **không có nền tảng code**
- Không dạy code, chỉ giúp hiểu **luồng chạy + các vấn đề giải quyết được**
- Focus vào tư tưởng + phương pháp, không phải step-by-step
- Kết hợp Claude Code + MiniMax + đa AI model
